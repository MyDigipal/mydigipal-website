Les concessions automobiles passent à côté d'opportunités majeures. Pendant que vos concurrents se battent sur des mots-clés génériques à 30-80€ par lead, les dealers malins génèrent des visites en showroom à **5-15€ par lead** grâce à Google Vehicle Ads (GVA).

Selon la [documentation officielle Google](https://support.google.com/merchants/answer/11189169), les Vehicle Ads vous permettent d'atteindre les acheteurs auto au bon moment, avec les bonnes annonces pour les rapprocher de l'achat.

Ce guide complet couvre tout ce qu'il faut savoir pour dominer Google Vehicle Ads en 2026.

## Qu'est-ce que Google Vehicle Ads ?

Google Vehicle Ads affiche automatiquement votre inventaire dans les résultats de recherche Google. Contrairement aux annonces texte classiques, elles montrent :

| Élément | Ce que l'utilisateur voit |
|---------|---------------------------|
| Image véhicule | Photo haute qualité du véhicule réel |
| Marque & modèle | "BMW X5 xDrive40i 2024" |
| Prix | Prix réel (conformité obligatoire) |
| Kilométrage | Pour les véhicules d'occasion |
| Nom concession | Votre marque |
| Localisation | Distance depuis l'utilisateur |

Quand un utilisateur cherche "BMW X5 près de chez moi" ou "SUV occasion moins de 30000€", votre inventaire réel apparaît.

## Configuration technique : Prérequis complets

### Prérequis compte

| Composant | Exigence | Priorité |
|-----------|----------|----------|
| Compte Google Ads | Actif, en règle | Obligatoire |
| Google Merchant Center | Configuré pour véhicules | Obligatoire |
| Fiche Google Business | Vérifiée, liée | Obligatoire |
| Vérification domaine | Domaine business vérifié | Recommandé |
| [Tracking conversions](/fr/services/tracking-analytics) | Correctement configuré | Obligatoire |

### Spécifications du flux véhicules

Votre flux inventaire est la base. Incluez ces champs obligatoires :

| Champ | Type | Exemple | Notes |
|-------|------|---------|-------|
| id | String | "VIN123456789" | Unique par véhicule |
| title | String | "BMW X5 xDrive40i 2024" | Année + Marque + Modèle + Finition |
| description | String | "Pack M Sport, toit panoramique..." | 150-500 caractères |
| availability | String | "in stock" | Doit être disponible |
| price | String | "65000.00 EUR" | Prix réel (conformité !) |
| link | URL | "https://concession.fr/bmw-x5-123" | Page détail véhicule |
| image_link | URL | "https://concession.fr/images/x5.jpg" | Image principale |
| brand | String | "BMW" | Constructeur |
| vehicle_id | String | "VIN123456789" | Numéro VIN |
| body_style | String | "SUV" | Type véhicule |
| mileage | Object | `{"value": 15000, "unit": "KM"}` | Obligatoire pour VO |

### Champs additionnels recommandés

| Champ | Impact sur performance |
|-------|------------------------|
| color | Améliore le matching pour recherches spécifiques |
| transmission | Filtre automatique vs. manuelle |
| fuel_type | Critique pour recherches VE/hybride |
| drivetrain | Requêtes 4x4/AWD spécifiques |
| condition | neuf/occasion/certifié |
| year | Année modèle |

### Exigences images

Google a des règles strictes :

| Exigence | Détails |
|----------|---------|
| Pas de surimpressions | Pas de texte, watermark, ou logos |
| Véhicule complet | La voiture entière doit être visible |
| Haute résolution | Minimum 800x600 pixels |
| Max images | 10 par véhicule |
| Pas de placeholder | Images "bientôt disponible" refusées |
| Angle 3/4 avant | Meilleure performance en image principale |

### Conformité page de destination

Vos pages détail véhicule (VDP) doivent afficher :

| Élément | Exigence |
|---------|----------|
| Nom concession | Clairement visible |
| Localisation | Adresse ou carte |
| Prix véhicule | **Doit correspondre exactement au flux** |
| Prix catalogue | Pour véhicules neufs |
| VIN | Visible sur la page |
| Kilométrage | Pour véhicules d'occasion |
| Disponibilité | En stock (pas vendu/réservé) |

## Règles de conformité prix (En vigueur)

La politique "Pratiques tarifaires trompeuses" de Google est maintenant appliquée :

### Exigences actuelles

| Règle | Détails |
|-------|---------|
| Exactitude prix | Prix affiché = prix d'achat réel |
| Inclusion frais | Tous les frais obligatoires inclus |
| Mises à jour temps réel | Changements de prix sous 24h |
| Pas d'appât | Prix page correspond à l'annonce |

### Conséquences des violations

| Gravité | Conséquence |
|---------|-------------|
| Première violation | Email d'avertissement, quelques jours pour corriger |
| Problèmes continus | Annonces suspendues jusqu'à correction |
| Répétées/graves | Suspension du compte |

**Recommandation** : Auditez régulièrement votre flux tarifaire. Assurez-vous que les prix de votre site correspondent exactement à ceux du flux.
